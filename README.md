# Consciousness Commons

This is the repository for the **Consciousness Commons Workshop 2026**, 
a two-day collaborative satellite workshop taking place July 4–5, 2026, 
following the Association for the Scientific Study of Consciousness 
conference ([ASSC 2026](https://theassc.org/assc-29/)).

The workshop brings together early-career researchers to develop new 
scientific projects using open datasets in consciousness science.
